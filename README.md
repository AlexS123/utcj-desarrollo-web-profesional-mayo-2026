## Dinámica de trabajo

Cada alumno trabajará dentro de su carpeta personal:

```txt
practicas/nombre-apellido/
```

Durante el curso se desarrollará:

- Un proyecto evolutivo principal
- Prácticas independientes según el tema visto en clase

---

## Ejemplo de estructura

```txt
practicas/
└── alumno/
    ├── proyecto-base/
    ├── practica-api/
    └── practica-react-router/
```
